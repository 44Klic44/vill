Task Manager — Управление задачами (MERN + Redux Toolkit)
Полнофункциональное веб-приложение для командной работы над задачами с современным адаптивным интерфейсом, ролевой моделью, системой уведомлений и дашбордом.

🚀 Технологический стек
Frontend

React 18 + Vite (быстрая сборка, HMR)

Redux Toolkit + RTK Query (управление состоянием, кэширование, инвалидация тегов)

React Router v6 (маршрутизация)

Tailwind CSS + Headless UI (адаптивный дизайн, доступные компоненты)

React Icons, Moment.js, Sonner (уведомления), React Hook Form

Полностью адаптивная вёрстка (mobile-first, кастомные хуки для медиа-запросов)

Backend

Node.js + Express

MongoDB + Mongoose (ODM)

JWT + httpOnly cookie (аутентификация)

bcryptjs (хеширование паролей)

Ролевая модель (администратор / пользователь)

✅ Реализованный функционал
🔐 Аутентификация и пользователи
Регистрация, вход, выход (JWT в httpOnly cookie)

Профиль пользователя (имя, роль, должность, email)

Администратор: управление пользователями (активация/деактивация, удаление)

Список команды с аватарами (инициалы)

📋 Задачи
Создание, редактирование, дублирование

Назначение исполнителей (мультивыбор)

Приоритеты (high, medium, low, normal) и статусы (todo, in progress, completed)

Изменение статуса через меню или перетаскивание (drag & drop)

Добавление подзадач и отметка выполнения

Комментарии/активности к задаче

Детальный просмотр (две вкладки: детали + таймлайн активностей)

Мягкое удаление (корзина), восстановление, полное удаление

🗑 Корзина (Trash)
Адаптивный интерфейс: на десктопе таблица, на мобиле карточки

Восстановление/удаление отдельной задачи

🔔 Уведомления
Автоматическое создание при назначении задачи

Попап с непрочитанными, отметка о прочтении (одно или все)

Детальный просмотр уведомления

📊 Дашборд
Карточки статистики (всего задач, выполнено, в процессе, todo)

График распределения по приоритетам (кастомный компонент Chart)

Последние 10 задач и последние пользователи (адаптивно: таблицы/карточки)

🎨 Интерфейс
Два представления задач: канбан-доска (Board View) и таблица (List View)

Быстрое создание задачи с нужным статусом (клик по заголовку колонки)

Меню действий (три точки): просмотр, редактирование, смена статуса, в корзину

Адаптация под мобильные устройства (карточки вместо таблиц, корректное позиционирование попапов)

Удалены все console.log для чистого продакшена

🛡 Безопасность
JWT в httpOnly cookie (защита от XSS)

Защищённые маршруты на клиенте и сервере

Ролевая проверка (админ / обычный пользователь)
